# blog-content

**When to use.** When the user runs (or wants to run) a long-form blog for owned-audience content marketing, distinct from SEO-targeted cluster pages.

**Procedure (stub).** Decide the blog's job: Think-stage education (operator-facing how-tos, frameworks), See-stage category-defining essays, or Care-stage customer-success stories. Pick one primary job and let the others occur opportunistically. Spec post length (long-form 1,500-3,000 words for Think and See stages), publishing cadence the team can sustain (1-2 posts/week is a common active floor; one substantive post/month is the sustainable minimum), and a distribution plan — a blog post without a distribution plan is a tree falling in an empty forest. Coordinate with the seo-organic skill for keyword-targeted pages; this skill covers narrative-led content. Hand writing to Copy; measure by time-on-page, scroll-depth, and downstream signup or purchase events.

**Status.** Default tier: user-toggle. Full procedure deferred to Phase 2. Until then, the channel-strategy skill handles cross-channel decisions.
