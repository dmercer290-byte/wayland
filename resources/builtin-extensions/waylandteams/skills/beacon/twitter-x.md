# twitter-x

**When to use.** When the channel mix includes X (formerly Twitter) for personal brand, B2B reach, or real-time category presence.

**Procedure (stub).** Assign X a stage — typically See (broad reach via reply-game and threads) or Think (long-form threads as comparison or framework content). Build a mix of standalone short posts (hook + payoff), threads (Think-stage framework explanations), and replies on high-traffic posts in the category (compounding distribution). Spec format constraints; hand wording to Copy. Measure by impressions, profile visits, and DM-initiation rate rather than likes.

**Status.** Default tier: user-toggle. Full procedure deferred to Phase 2. Until then, the channel-strategy skill handles cross-channel decisions.
